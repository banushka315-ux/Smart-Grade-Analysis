# Smart Grade Analysis

AI-powered Grade Analysis Web Application for subject-wise result analysis and grade segregation using Excel files.

## Features

- Upload Excel result sheets
- Subject-wise grade analysis
- Grade segregation (A+, A, B+, B, C+, C)
- Interactive charts and visualizations
- Student performance insights
- Clean and responsive user interface
- Fast and automated result processing

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- JavaScript
- HTML/CSS

## How It Works

1. Upload the Excel result file
2. Map subject codes with subject names
3. Analyze student grades automatically
4. Generate charts and statistics instantly

## Project Goal

The goal of this project is to simplify academic result analysis and reduce manual work for teachers, schools, and colleges.
